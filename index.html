Connect Google Drive
Your data will sync as pocketledger-data.json in your Drive. One-time setup takes ~3 minutes.
1
Open Google Cloud Console → create a new project (e.g. PocketLedger).
2
Go to APIs & Services → Library → search Google Drive API → Enable it.
3
Go to OAuth consent screen → choose External → fill App name → Save. Add your Google email as a Test user.
4
Go to Credentials → Create Credentials → OAuth Client ID → type: Web application.
Add this under Authorized JavaScript origins:
https://zahir049.github.io/pocketledger/
5
Copy the Client ID (ends in .apps.googleusercontent.com) and paste below.
6
Serve this file with: npx serve . — then open https://zahir049.github.io/pocketledger/
